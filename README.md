## 📱 إضافة إلى الشاشة الرئيسية (Mobile)

### Android (Chrome):
1. افتح الموقع في Chrome
2. اضغط على القائمة (⋮)
3. اختر **"إضافة إلى الشاشة الرئيسية"** أو **"تثبيت التطبيق"**
4. اضغط **"تثبيت"**

> ⚠️ **إذا لم يظهر الخيار:** جرب مسح بيانات Chrome من إعدادات الجهاز

### iPhone (Safari):
1. افتح الموقع في Safari
2. اضغط على زر المشاركة (□↑)
3. اسحب للأسفل واختر **"إضافة إلى الشاشة الرئيسية"**
4. اضغط **"إضافة"**

### Samsung Internet:
1. اضغط على القائمة (☰)
2. اختر **"إضافة صفحة إلى"** → **"الشاشة الرئيسية"**

## 🌐 النشر على Render (مجاني)

### الخطوة 1: رفع على GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/USERNAME/institute-schedule-pwa.git
git push -u origin main
```

### الخطوة 2: النشر على Render
1. سجل في [render.com](https://render.com) بحساب GitHub
2. اضغط **New Web Service**
3. اختر مستودع `institute-schedule-pwa`
4. Render يكتشف الإعدادات تلقائياً من `render.yaml`
5. اضغط **Deploy**

بعد 2-3 دقائق يكون التطبيق على الإنترنت:
```
https://institute-schedule.onrender.com
```

## 🔧 حل مشكلة Chrome على Android

إذا لم يظهر خيار "إضافة إلى الشاشة الرئيسية":

### الحل 1: مسح بيانات Chrome
```
الإعدادات ← التطبيقات ← Chrome ← التخزين ← مسح البيانات والذاكرة المؤقتة
```

### الحل 2: استخدام Samsung Internet
```
افتح الموقع في Samsung Internet ← القائمة ← إضافة صفحة إلى ← الشاشة الرئيسية
```

### الحل 3: نسخ الرابط
```
انسخ الرابط ← أرسله لنفسك على WhatsApp ← افتحه من هناك ← جرب التثبيت
```

### الحل 4: تفعيل "إضافة التطبيقات الجديدة"
```
الإعدادات ← الشاشة الرئيسية ← تفعيل "إضافة التطبيقات الجديدة"
```
